# Design (`docs/design/`)

Phase-2 deliverable. Numbered design specs that form the permanent citation address space the
backlog points at (`architecture.md §4.2`). The design docs are the single source of truth;
backlog tasks are thin scope wrappers that reference these sections — they never restate design
content.

Each design doc:
- starts with the SPDX/license header, is markdown-linted, and numbers every section;
- carries trace-or-deviate citations to `docs/research/` for every substantive claim;
- tags every invented constant `(PI)` and points at the central calibration table.

> **Status:** populated in Phase 2 (architecture), after the agent-panel ADRs are accepted.
