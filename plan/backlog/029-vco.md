## Objective

Implement the Oscillator master phase accumulator, the exponential 1V/oct converter, the 16'/8'/4'/2' footage CV offsets, the dt clamp, and the small drift/stability model (hooks present, default near-zero).

## Context

- `01-dsp-oscillators.md §4.1-4.4` — read first
- `01-dsp-oscillators.md §4.7` — read first
- `01-dsp-oscillators.md §2.1` — read first
- `01-dsp-oscillators.md §10 (footage offsets, real-time safety)` — read first
- `plan/ORCHESTRATION.md` — owner-locked decisions; `AGENTS.md` — operating rules (scope, TDD, silent-pass).
- TDD: write the failing test(s) first under `tests/`; test-case names begin with `vco`.

## Scope

- Create core/dsp/Oscillator.h/.cpp scaffolding: OscControls struct, prepare(sampleRate, hqTable), reset(), setControls(), phase()/wrappedThisSample()/frequencyHz()
- Exp pitch: freqHz = kPitchRefHz*2^(pitchCvVolts - kPitchRefVolts + footageOffsetV + driftScale); constants from Calibration.h
- Footage octave offsets 16'/8'/4'/2' = -1/0/+1/+2 about 8' reference; dt_ = min(freqHz/fs, kDtMax)
- Drift/stability model (warm-up settle, scale/offset error, HF-track flag) reading kDrift*/kWarmupTauSec/kHfTrackEnable from Calibration.h; expose scaleErr_/offsetErr_ seeds
- renderSample() advances master phase once per sample (waveform output stubbed/raw ramp; band-limiting added in core-osc-4)

## Out of scope

- Band-limited saw/pulse construction and PWM mapping (core-osc-4)
- Sub-oscillator division (core-osc-5)
- Tune/bender/LFO CV summation (owned by control-core/env-lfo-vca; VCO consumes pre-summed volts)

## Acceptance criteria

- [ ] vco-* tests assert footage 4' is exactly 2x the 8' frequency etc., and 8'+Transpose-Middle+0-cent = 442.0 Hz [§4.3, §4.4, §10]
- [ ] dt_ never exceeds kDtMax=0.5 (at most one wrap/sample) across the audio band [§4.4]
- [ ] every (PI) pitch/footage/drift constant is referenced from Calibration.h, not duplicated [§10]
- [ ] renderSample/phase/frequencyHz are noexcept with no heap alloc or locks (guard) [§2.4, §10]
